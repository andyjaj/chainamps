Detailed Description

MPX_matrix is the base class for MPS and MPO matrices.

Defines the tensor class and related objects.

Essentially a container for a SparseMatrix, with lots of bells and whistles so that they can be blocked by good quantum numbers, and reshaped according to their indices.
