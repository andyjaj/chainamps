Detailed Description

State is a container for some good quantum numbers, and the rules used to combine them.

States can be added and subtracted, using the CHARGE_RULES to decide the correct modulo arithmetic. Adding State objects is equivalent to working out the resulting total charges for the tensor product of two actual objects. Subtracting states is useful when one considers directed arrows in MPS contractions.
